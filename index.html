WEB DEVELOPMENT

HTML, CSS, and JavaScript are foundational technologies used for creating websites and web applications. Here's a brief overview of each:

  1. HTML (HyperText Markup Language):
    • HTML is the standard markup language for creating the structure and content of web pages.
    • It uses a system of tags and elements to define different parts of a web page, such as headings, paragraphs, images, links, and more.
    • HTML provides the basic structure that is used by web browsers to render web pages correctly.
  2. CSS (Cascading Style Sheets):
    • CSS is used to style the layout and appearance of web pages created with HTML.
    • It allows you to control the colors, fonts, spacing, and overall design of a web page.
    • CSS uses selectors to target specific elements on a web page and apply styles to them.
  3. JavaScript:
    • JavaScript is a programming language that adds interactivity and dynamic behavior to web pages.
    • It can be used to create interactive features like dropdown menus, form validation, animations, and more.
    • JavaScript is executed by the web browser, making it possible to create responsive and interactive web applications.

HTML:

TAG AND ATTRIBUTE:

Tags in HTML are keywords enclosed in angle brackets (<>) used to define elements on a web page. There are two types of tags:

  • Void Tags: These are self-closing tags that don't require a closing tag, like <img> or <br>.
  • Opening and Closing Tags: These come in pairs, with the content between them. Examples include <p> for paragraphs or <div> for divisions.

Attributes provide extra information about an element and are added to the opening tag. They're written as name="value". For example, href="https://example.com" in <a href="https://example.com">Link</a>.

Understanding tags and attributes is essential for creating structured and visually appealing web pages.

IMPORTANT TAGS:

Here's a brief explanation of each tag along with an example:

  1. Heading Tags (<h1>, <h2>, <h3>, <h4>, <h5>, <h6>):
    • Heading tags are used to define headings on a web page.
    • They range from <h1> (the highest level) to <h6> (the lowest level).
    • Example: <h1>This is a Heading</h1>
  2. Paragraph Tag (<p>):
    • The paragraph tag is used to define paragraphs of text.
    • Example: <p>This is a paragraph.</p>
  3. Horizontal Rule Tag (<hr>):
    • The horizontal rule tag is used to create a horizontal line on the page.
    • Example: <hr>
  4. Break Tag (<br>):
    • The break tag is used to create a line break within text.
    • Example: First line<br>Second line
  5. Ordered List Tag (<ol>, <li>):
    • The ordered list tag is used to create a numbered list.
    • Example: 
      <ol>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ol>
  6. Unordered List Tag (<ul>, <li>):
    • The unordered list tag is used to create a bulleted list.
    • Example: 
      <ul>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ul>
  7. Nested List:
    • Lists can be nested inside one another to create hierarchical structures.
    • Example: 
      <ul>
  <li>First item</li>
  <li>Second item
    <ul>
      <li>Nested item 1</li>
      <li>Nested item 2</li>
    </ul>
  </li>
  <li>Third item</li>
</ul>
  8. Anchor Tag (<a>):
    • The anchor tag is used to create hyperlinks.
    • Example: <a href="https://example.com">Link</a>
  9. Image Tag (<img>):
    • The image tag is used to display images on a web page.
    • Example: <img src="image.jpg" alt="Image Description">

DIFFERENCE BETWEEN ‘src’ AND ‘href’

  1. src:
    • The src attribute is used in tags like <img>, <script>, and <iframe> to specify the source (URL) of the resource being embedded.
    • It is used to load external content into the current document, such as images, scripts, or other HTML documents.
    • Example: <img src="image.jpg">
  2. href:
    • The href attribute is used in the <a> (anchor) tag to specify the destination (URL) of the link.
    • It is used to create hyperlinks, allowing users to navigate to different pages or resources.
    • Example: <a href="https://example.com">Link</a>

PATH:

In computing, a path is a unique location to a file or directory in a file system of an operating system. Paths are used to specify the location of files and directories when referencing them in commands or programs. There are two main types of paths:

  1. Relative Path:
    • A relative path specifies the location of a file or directory relative to the current working directory.
    • Relative paths do not begin with a root (/ in Unix-like systems or .\User\admin in Windows) and are used to navigate within the current directory or to files and directories nearby.
    • Examples:
      • If the current directory is /home/user/ and you want to access a file named example.txt located in the same directory, the relative path would be example.txt.
      • If you want to access a file named file.txt located in a subdirectory named docs within the current directory, the relative path would be docs/file.txt.
  2. Absolute Path:
    • An absolute path specifies the exact location of a file or directory in the file system from the root directory.
    • Absolute paths begin with the root directory and include all directories leading to the target file or directory.
    • Examples:
      • In Unix-like systems, an absolute path might look like /home/user/example.txt.
      • In Windows, an absolute path might look like C:\Users\User\Documents\file.txt.

STRUCTURE OF HTML:

The basic structure of an HTML5 document consists of the following elements:

  1. Document Type Declaration: <!DOCTYPE html> is used to specify that the document is an HTML5 document.
  2. HTML Element: The <html> element is the root element of an HTML page. It contains two main sections: the <head> section and the <body> section.
  3. Head Section: The <head> section contains meta-information about the document, such as the title of the document, links to external stylesheets, and metadata.
  4. Body Section: The <body> section contains the main content of the document, such as text, images, links, and other elements that are displayed on the web page.

The basic structure of an HTML5 document looks like this:

<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Title of the Document</title>
</head>
<body>
    <header>
        <!-- Header content -->
    </header>
    <nav>
        <!-- Navigation menu -->
    </nav>
    <main>
        <!-- Main content -->
    </main>
    <footer>
        <!-- Footer content -->
    </footer>
</body>
</html>
